23. Заряд плоского конденсатора с диэлектриками равен 8,85⋅$10^{-8}$ Кл. Площадь каждой пластины 50 $см^2$. Если диэлектрическач проницаемость среды равна 2, то напряженность поля в диэлектрике ($ε_0 = 8,85⋅10^{-12} Ф/м$ )

A) 5⋅$10^6$ В/м
B) 2⋅$10^6$ В/м
C) 3⋅$10^6$ В/м
D) $10^6$ В/м
E) 4⋅$10^6$ В/м

Ответ: D

Заряд конденматора Q = CU=> U = Q/C. C = $εε_0S/d$, где ε - диэлектрическая проницаемость среды, S - площадь каждой пластины, d - расстояние между пластинами.
U = $Q/{εε_0S/d}$
Отсюда E = $U/d$ = $Q/{εε_0S}$ = ${8,85⋅10^{-8}}/{2⋅8,85⋅10^{-12}⋅50⋅10^{-4}}$ = $10^6$ В/м.
