19. Эксперименты по термоэлектронной эмиссиии фотоэффекту свидетельствуют о том, что в состав

A) атома входят электроны.
B) атома входят нейтроны.
C) ядра входят протоны и нейтроны.
D) атома входят только протоны.
E) атома входят протоны.

Ответ: A

Решение: Фотоэффект - это явление испускания электронов веществом под действием света.
